A fully responsive Spotify UI clone created using pure HTML and CSS.
This project recreates the visual interface of Spotify, including the sidebar, navigation, playlists, cards, and bottom music player.

-Features
i) Modern Spotify-inspired design
ii) Sidebar navigation with Home, Search & Library
iii) Horizontal scrollable sections (Recently Played, Trending, etc.)
iv) Bottom music player UI
v) Custom Spotify-style scrollbar using standart web toolkits
vi) Hover animations and smooth transitions
vii) Responsive layout for different screen sizes

-Built With
i) HTML5
ii) CSS3
